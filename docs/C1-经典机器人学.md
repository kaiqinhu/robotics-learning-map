## 2. 运动学 (Kinematics)

### 正向运动学：关节角 → 末端位姿

```
给定：关节角 q = [q₁, q₂, ..., qₙ]
求：末端位姿 T_ee ∈ SE(3)

T_ee = FK(q) = T₁(q₁) · T₂(q₂) · ... · Tₙ(qₙ)

每个 Tᵢ(qᵢ) 是一个 4×4 的齐次变换矩阵（SE(3) 元素）。
对于旋转关节：Tᵢ(qᵢ) = exp(qᵢ · ξᵢ)   ← 指数映射！SE(3) 的 exp 在这里就是物理运动
```

**关键直觉：** 机器人运动学就是 SE(3) 指数映射的连乘。你学 SE(3) 不是为了数学，是因为机械臂天生活在 SE(3) 里。

### 逆向运动学：末端位姿 → 关节角

```
给定：目标末端位姿 T_target ∈ SE(3)
求：关节角 q 使得 FK(q) = T_target

→ 通常没有解析解，用数值迭代（牛顿法）
→ 核心工具：雅可比矩阵 J
```

---

## 3. 雅可比矩阵 (Jacobian)

### 几何雅可比

```
J(q) ∈ R^{6×n}

作用 1（速度层面）：
  [ω, v]ᵀ = J(q) · q̇
  关节速度 q̇ → 末端 twist（旋转速度 ω + 平移速度 v）∈ se(3)

作用 2（力层面）：
  τ = J(q)ᵀ · [τ, f]ᵀ
  末端力/力矩 wrench → 关节力矩 τ

  这就是阻抗控制 τ = Jᵀ(K·Δx + B·Δẋ) 的核心！
```

### 为什么 J 是核心？

```
运动方向：q̇  →(J)→  末端运动 [ω,v] ∈ se(3)
力方向：  末端力 F →(Jᵀ)→ 关节力矩 τ

这两个是同一个 J，只是转置方向不同（对偶性）。
```

## 4. 动力学 (Dynamics)

### 运动方程

```
M(q) · q̈ + C(q,q̇) · q̇ + g(q) = τ_motor + J(q)ᵀ · F_ext

M(q)：    质量矩阵（惯性）
C(q,q̇)：  科里奥利力 + 离心力
g(q)：    重力项
τ_motor： 电机力矩（你控制的）
F_ext：   外部接触力（环境施加的）
Jᵀ·F_ext：外部力"反射"到关节空间
```

### 力控中的意义

```
τ_control = τ_motor = Jᵀ · F_desired + g(q)  ← 重力补偿
                       + Jᵀ · K · (x_d - x)   ← 阻抗项
                       + Jᵀ · B · (ẋ_d - ẋ)   ← 阻尼项

如果你不知道 g(q)（重力），你的力控会在重力方向偏一个常值。
如果你不知道 M(q)，大加速度时力控会失准。
```

## 5. 力控中常用到的几个变换

```
末端 wrench  → 关节力矩：  τ = J(q)ᵀ · F
关节力矩     → 末端力：    F = J(q)^{-T} · τ  （J 可逆时）

关节阻抗 → 末端阻抗：
  K_x = J^{-T} · K_q · J^{-1}  （关节刚度 → 末端刚度）

末端阻抗 → 关节阻抗：
  K_q = Jᵀ · K_x · J  （末端刚度 → 关节刚度）
```

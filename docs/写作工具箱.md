# ✍️ 写作工具箱

> **目的：** 你能一目十行地读，但写东西需要训练。这里提供一套从「填空」到「自由输出」的渐进式写作方法，嵌入在每个专题笔记的末尾。
>
> **用法：** 学完任何一个专题的概念部分后，按 L1 → L2 → L3 的顺序做写作练习。不要跳过 L1——它是 L3 的脚手架。

---

## 为什么读得快但写得差？

```
阅读是「识别」—— 你看到别人的逻辑链，顺着走就行
写作是「构建」—— 你要从零开始搭一条逻辑链

这是两种完全不同的认知能力。
阅读能力强 ≠ 写作能力强

解决方式：
  用「结构化输出」替代「空白页恐惧」
  → 给你模板、给你开头、给你约束
  → 把「写一篇文章」拆成「回答 3 个问题」
```

---

## L1：费曼笔记法（检验是否真懂）

### 规则

用一段话解释一个概念，**假设读的人完全不懂这个领域**。不允许用术语，不允许跳步骤。

### 模板

```markdown
## [概念名] — 费曼笔记

**一句话：** [用最简单的话说这是什么]

**类比：** [用一个日常生活的类比来解释，如"这就像..."]

**为什么不用别的方法？** [这个方法解决了什么问题，为什么别的方法不行]

**我最初哪里理解错了：** [写下你一开始的误解，以及后来怎么纠正的]
```

### 例子

```markdown
## Flow Matching — 费曼笔记

**一句话：** 给每个噪声样本指定一条"水流"，让它沿着水流漂到目标位置。

**类比：** 类似于你要把 100 个散落在操场上的学生聚集到教室——
你给每个学生画一条箭头，告诉他们"沿着箭头走"。FM 就是学"怎么画这些箭头"。

**为什么不用别的方法？** Diffusion 相当于让学生随机漫步，走 1000 步
才能到教室。FM 让学生走直线，10 步就到。

**我最初哪里理解错了：** 我以为 FM 和 Diffusion 是一回事，只是换个名字。
后来才理解 FM 学的是确定性向量场，Diffusion 学的是随机过程的 score
——数学上等价但实际实现完全不同。
```

## L2：概念串联（建立知识网络）

### 规则

画一张图（文字图即可），把本专题的 3-5 个核心概念串联起来。

### 模板

```markdown
## [专题名] — 概念地图

用箭头标注因果关系：

[概念 A]
  │
  ├── 因为 A 有 X 问题 ──→ [概念 B] 解决了这个问题
  │
  └── A 和 C 结合 ──→ [概念 D] 是我可以用的设计

给每条箭头写一句话解释：
- A → B：[一句话]
- A+C → D：[一句话]
```

## L3：设计性写作（创造新东西）

### 规则

以"如果我设计..."开头，写一个简短的、具体的方案。

### 模板

```markdown
## 如果我设计 [某个系统/模型]...

**场景：** [具体的应用场景]

**约束：** [你必须面对的限制——算力、延迟、数据量、传感器等]

**我的方案：**

1. 输入是什么：[...]
2. 输出是什么：[...]
3. 核心架构选择：[为什么选 X 而不是 Y]
4. 最可能失败的地方：[...]
5. 我打算怎么验证：[...]

**如果不考虑工程约束，理想方案是什么？**

[...]
```

### 为什么 L3 最重要？

L1 和 L2 是「理解」，L3 是「创造」。
你的最终目标是设计自己的模型——L3 就是这种能力的训练。
每次学完一个专题，做一次 L3，你就在积累「设计直觉」。

## 写作节奏建议

```
每学完一个专题的一个小节：
  → 做 1 次 L1（5 分钟，写在小节末尾）

每学完一个完整专题：
  → 做 1 次 L2（15 分钟）
  → 做 1 次 L3（30 分钟，可以草稿，后续迭代）

学完 3 个相关专题后：
  → 做 1 次跨专题 L3（1 小时，如"如果我设计 SE(3)+力控的联合 FM..."）

每月：
  → 回看自己写的 L3，用新学的知识修改
  → 这是最好的进步度量——你会看到一个月前自己想的有多粗糙
```

## 常见写作问题诊断

| 症状 | 诊断 | 补救 |
|------|------|------|
| "不知道从哪里开始写" | 缺少框架 | 先用 L1 的填空模板 |
| "写了但很乱，没有逻辑" | 跳过 L2 直接写 L3 | 先画概念图，再写文字 |
| "写出来的都是别人的话" | 没有用自己的类比 | 强制要求每段有一个"就像..." |
| "写到一半放弃了" | 完美主义 | 先写烂，再改。标题写 [草稿] |
| "不知道写得对不对" | 缺少反馈 | 把你写的 L1 给不懂的人看，看他能不能懂 |

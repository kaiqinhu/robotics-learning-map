## 背景：R^6 是一个陷阱

你可能直觉上觉得：位姿 = 3 个位置 + 3 个角度 → 6 个数 → 在 R^6 里做 FM 就行了。

**不行。** 因为旋转不是向量——旋转矩阵的列正交、行列式为 1。在 R^6 里做线性插值，你会得到行列式 ≠ 1 的"旋转"——物理上根本不存在。在 R^6 里做高斯采样、做向量场——你不知道这些操作在旋转空间里意味着什么。

**SE(3) 是一个弯曲的 6 维流形，不是平的 R^6。** 你需要 exp/log 映射来在流形和切空间之间来回，正如同你用经纬度在地球表面和平面地图之间来回。在 M1 里，你的 FM 是在 SE(3) 上生成动作——如果你不知道 exp/log 是什么，你就不知道为什么 M1 的损失函数里有一个黎曼度量而不是 L2。

本页不讲证明——只讲直觉和会用的公式。目标是：看到 `exp(log(T1) + t*log(T2))` 这个公式时，你知道它在干什么。

## 学完这一页你能

- 解释"为什么位姿不在 R^6 里"——用旋转矩阵的约束（RᵀR = I, det(R) = 1）
- 用指数映射把切空间向量变成 SE(3) 元素，用对数映射反过来
- 理解测地线：SE(3) 上的"直线"是什么？和 R^d 里的线性插值有什么不同？
- 知道为什么在 R^6 里做 FM 生成位姿会出问题，以及在 SE(3) 上做 FM 需要改什么（预演 M1）

## 1. 问题：R^6 里的"位姿"其实是错的

```
常见错误做法：
  pose = (x, y, z, roll, pitch, yaw) ∈ R^6
  在 R^6 里做插值、做概率分布、做 FM

问题：
  1. 两个旋转之间 L2 距离 ≠ 旋转角度距离
     (roll=179°, pitch=0°,  yaw=0°) 和 (roll=-179°, pitch=0°, yaw=0°)
     → 几乎一样的姿态，L2 距离却巨大

  2. 线性插值两个旋转矩阵 → 结果不在 SO(3)（行列式 ≠ 1）
     R_interp = 0.5*R₁ + 0.5*R₂ → R_interp 不是正交矩阵

  3. 欧拉角有万向节锁——不是所有姿态都能唯一表示
```

## 2. SO(3): 旋转空间

### 几何直觉

```
SO(3) 可以想象成 4 维球面 S³ 的一半。
    → 3 维的旋转空间嵌入在 4 维空间里
    → 它是一个弯曲的、紧致的 3 维流形

单位四元数表示 SO(3) 最自然：
  q = [w, x, y, z],  w²+x²+y²+z²=1
  q 和 -q 表示同一个旋转（对径点等价 → S³ 的一半）
```

### 核心运算

```
so(3): 3×3 反对称矩阵（"旋转速度"的空间，3维向量空间）

指数映射 exp: so(3) → SO(3)
  exp(ω̂) = I + sin(θ)/θ · ω̂ + (1-cos(θ))/θ² · ω̂²
  → 物理意义：绕轴 ω 转 θ 弧度

对数映射 log: SO(3) → so(3)
  log(R) = θ/(2sinθ) · (R - Rᵀ)
  → 物理意义：从旋转矩阵中提取"旋转了 θ 弧度，绕轴 ω"

测地线距离（两个旋转之间的"最短角距离"）：
  d(R₁, R₂) = || log(R₁ᵀ · R₂) ||
```

## 3. SE(3): 刚体运动空间

### 表示

```
SE(3) 是 6 维流形：3 维旋转 × 3 维平移（但它们是耦合的——半直积）

齐次矩阵表示：
  T = [R  t]   R ∈ SO(3), t ∈ R³
      [0  1]

se(3): 6 维向量空间（"运动旋量"的空间）
  ξ = [ω, v]   ω ∈ so(3)（旋转速度），v ∈ R³（平移速度）
```

### 核心运算

```
exp: se(3) → SE(3)
  把"小运动旋量"变成刚体变换矩阵

log: SE(3) → se(3)
  把刚体变换矩阵变成运动旋量

测地线插值（SE(3) 上的"直线运动"）：
  T(τ) = T₀ · exp(τ · log(T₀⁻¹ · T₁))

  → 这保证插值结果始终在 SE(3) 上
  → 旋转走最短弧，平移走直线
  → 这是你 FM 条件路径的基础

测地线距离（需要加权——弧度和米量纲不同）：
  d(T₁, T₂) = || log(T₁⁻¹ · T₂) ||_W
  ||(ω, v)||²_W = ||ω||²/σ_r² + ||v||²/σ_t²
```

## 4. 对你 FM 设计的影响

```
在 R^d 里做 FM：
  x_t = (1-t)·x₀ + t·x₁         ← 线性插值
  v_target = x₁ - x₀             ← 恒定速度

在 SE(3) 上做 FM：
  T_t = T₀ · exp(t · log(T₀⁻¹·T₁))   ← 测地线插值
  v_target = log(T_t⁻¹·T₁)/(1-t)      ← 在 se(3) 中的条件向量场

你的网络输出：不是 R^6 向量，而是 se(3) 中的切向量。
损失函数中的距离：不是 L2，是 SE(3) 的加权测地线度量。
```

# 🗺️ 具身操作基础模型 — 学习地图 v2

> **目标：** 成为具身模型/具身控制专家。能设计、实现、评估面向机械臂 SE(3) 位姿 + 力触觉 + 灵巧手的预训练与后训练模型。
>
> **核心理念：** 螺旋学习，不是线性推进。同一个概念（如 FM、SE(3)、力控）在 Core → Methods → Integration 三层以递进深度出现。
>
> **最后更新：** 2026-06-06

## ✍️ 写作训练体系

每篇 Core/Methods 文档末尾有 L1/L2 练习。**L3 统一写在 [I4-累积设计文档](/I4-累积设计文档) 里**，边学边迭代同一份方案：

```
Phase C: L3 → I4 section 1: 问题定义 + 输入/输出
Phase M1: L3 → I4 section 2: SE(3) 生成架构
Phase M2: L3 → I4 section 3: 多模态融合设计
Phase M3: L3 → I4 section 4: 触觉+力控方案
Phase M4+M5: L3 → I4 section 5: 训练范式

最终产物：一份经过 5 轮迭代的完整模型设计
```

## 🧭 核心问题清单

### 基础
- [ ] 为什么 SE(3) 不是 R^6？在 R^6 里做位姿插值会发生什么？
- [ ] FM 的 ODE 和 Diffusion 的 SDE 本质区别在哪？
- [ ] 雅可比矩阵 J 和 Jᵀ 分别出现在什么场景？

### 方法
- [ ] Riemannian FM 和普通 FM 的损失函数差在哪里？
- [ ] MoE router 的 load balancing 为什么是必须的？
- [ ] 触觉的 pretraining 能不能跨传感器迁移？

### 集成
- [ ] 大脑(FM 10Hz) 和小脑(阻抗控制 1000Hz) 之间传什么信号？
- [ ] UMI 数据和真机数据的位姿标签噪声性质有什么不同？
- [ ] 怎么评估一个力控策略的好坏？（不只是成功率）

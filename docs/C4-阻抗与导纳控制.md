# 🔴 C4-阻抗与导纳控制

> **核心问题：** 力控不是"预测一个力然后输出"——它是一条闭环。阻抗控制让机器人像弹簧一样应对外力。这是你力触觉策略的理论根基。
>
> **预计时间：** 1 周
>
> **状态：** 📝 待学习

---

## 1. 位置控制的局限

```
纯位置控制的假设：
  "机器人是刚体，环境是刚体，轨迹精确执行"
  
现实：
  → 机器人有柔性（关节弹性、连杆变形）
  → 环境有不确定性（物体位置误差 >1mm）
  → 接触力会突变（撞到东西 → 力瞬间飙升）
  
纯位置控制遇到误差 → 要么撞坏物体，要么撞坏自己
力控 → "顺应"误差，用柔顺性吸收不确定性
```

## 2. 阻抗控制 (Impedance Control)

### 物理直觉

```
把机器人末端想象成一个"弹簧-阻尼-质量"系统：

  F_ext = M · (ẍ - ẍ_d) + B · (ẋ - ẋ_d) + K · (x - x_d)

  如果外部施加一个力 F_ext：
    → 弹簧 K 决定"推多远才到位移"
    → 阻尼 B 决定"振动多久停下来"
    → 质量 M 决定"惯性有多大"

调参直觉：
  K 大 → 机器人"硬"，严格跟踪轨迹。适合自由空间运动
  K 小 → 机器人"软"，顺应外力。适合接触操作
  B 大 → 快速收敛，不震荡
```

### 关节空间实现

```
τ = J(q)ᵀ · [K · (x_d - x) + B · (ẋ_d - ẋ)] + g(q)
     ↑                          ↑                    ↑
  C1的雅可比           C3的刚度+阻尼             C1的重力补偿

如果没有重力补偿 g(q)，机器人在重力方向会下垂一个恒定位移。
```

## 3. 导纳控制 (Admittance Control)

### 阻抗 vs 导纳

```
阻抗控制：
  输入：位姿误差 Δx
  输出：力修正 ΔF
  → "你推我，我给你一个额外的力"
  → 适用：力传感器不在末端时（内环力矩控制）

导纳控制（对偶）：
  输入：测到的力 F
  输出：位姿修正 Δx = F / (M·s² + B·s + K)
  → "你推我，我就顺着动"
  → 适用：末端有力传感器时
```

### 什么时候用哪个？

```
有力/力矩传感器在末端 → 导纳控制（直接测外力 → 动）
没有末端力传感器       → 阻抗控制（用关节力矩估计外力 → 加力）

Franka, KUKA iiwa 有内置关节力矩传感器 → 多用阻抗控制
普通工业机器人加装 ATI 传感器 → 多用导纳控制
```

## 4. 混合力/位控制 (Hybrid Force/Position)

### 核心思想

```
某些方向用力控，某些方向用位控。

擦桌子：
  → 垂直桌面（z）：力控，恒压 5N
  → 平行桌面（x,y）：位控，按轨迹走

实现：
  τ = Jᵀ · [S · F_d + (I-S) · K_p · (x_d - x)]
            \________/   \_______________________/
             力控方向            位控方向
  
  S 是选择矩阵：S_ii = 1 表示该方向用力控，0 表示位控
```

## 5. 对你 FM 策略的影响

```
学习力控 = 学会调节 (K, B, x_d, F_d)

传统做法：工程师手工调 K 和 B
你的方向：FM 策略输出 (x_d(t), F_d(t), K(t), B(t))

  → 预接触阶段：K 大（硬），精确定位
  → 接触瞬间：K 突变小（软），切换到力控
  → 操作中：K 和 B 根据触觉反馈动态调整

FM 生成的是"阻抗参数 + 力目标 + 位姿目标"的联合轨迹
```
